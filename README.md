## 🚀 Cara Penggunaan

```bash
# Clone repository
git clone https://github.com/Strangersxyz/kelompok-4.git

# Masuk ke direktori project
cd kelompok-4

# Install dependencies
npm install

# Jalankan development server
npm run dev
